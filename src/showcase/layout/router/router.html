The router (aurora-router) contains the links to the individual sub-pages. This component can be used in combination with other components, such as the Accordion component. In this combination, you can build a menu like the one in this showcase. The router links (aurora-router-link) belong to the router, they represent the individual links. Eine gesamte Card kann als interaktive Fläche angelegt werden, wenn sie sonst keine interaktiven Elemente enthält.
