Der Router (aurora-router) enthält die Verlinkungen zu den einzelnen Unterseiten. Diese Komponente kann in Verbindung mit anderen Komponenten genutzt werden, wie zum Beispiel der Accordion-Komponente. In dieser Kombination kann man ein Menu wie in diesem Showcase aufbauen. Zum Router gehören die Router-links (aurora-router-link), sie bilden die einzelnen Links ab. Eine gesamte Card kann als interaktive Fläche angelegt werden, wenn sie sonst keine interaktiven Elemente enthält. Sie erhält dann einen Hover-Effekt. <aurora-router route-outlet="#main"> </aurora-router>
Property Type Default Value Description
route-outlet String - Hier wird die "id" des Elements angegeben, in dem der Inhalt der seite angezeigt werden soll.
